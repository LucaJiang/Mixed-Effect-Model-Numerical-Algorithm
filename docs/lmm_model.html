From OLM to LMM

The OLM (Ordinary Linear Model) assumes that the observations are independent and identically distributed (i.i.d.). It can be written as
y=Xγ+e,\begin{equation} \mathbf{y} = \mathbf{X}\mathbf{\gamma} + \mathbf{e}, \end{equation}y=Xγ+e,​​
where y∈Rn\mathbf{y} \in \mathbb{R}^ny∈Rn is the vector of response variable, X∈Rn×p\mathbf{X} \in \mathbb{R}^{n \times p}X∈Rn×p is the matrix of ppp independent variables, γ∈Rp\mathbf{\gamma} \in \mathbb{R}^pγ∈Rp is the vector of coefficients, and e∼N(0,σe2In)\mathbf{e} \sim \mathcal{N}(\mathbf{0}, \sigma^2_e \mathbf{I}_n)e∼N(0,σe2​In​) is the independent noise term. To simplify the notation, we assume that X\mathbf{X}X and y\mathbf{y}y are centered, i.e. the mean of y\mathbf{y}y is 0\mathbf{0}0, and the mean of X\mathbf{X}X is 0\mathbf{0}0 as well. Thus, we do not need to estimate the intercept term.

In the OLM, if we want to obtain the coefficients γ\mathbf{\gamma}γ, we can use the least square method to minimize the loss function. However, when n<pn<pn<p, the matrix X\mathbf{X}X is not full rank, and the least square method could lead to overfitting. Because we have p+1p+1p+1 (including σe2\sigma_e^2σe2​) parameters to estimate, but only nnn observations.

Let's assume the design matrix X\mathbf{X}X can be decomposed into two parts, i.e., X=[X1,X2]\mathbf{X} = [\mathbf{X}_1, \mathbf{X}_2]X=[X1​,X2​], where X1∈Rn×p1,p1<n≪p\mathbf{X}_1 \in \mathbb{R}^{n \times p_1}, p_1 < n\ll pX1​∈Rn×p1​,p1​<n≪p and X2∈Rn×p2\mathbf{X}_2 \in \mathbb{R}^{n \times p_2}X2​∈Rn×p2​. And, γ\mathbf{\gamma}γ can be break down into a fixed effect part ω∈Rp1\mathbf{\omega}\in \mathbb{R}^{p_1}ω∈Rp1​ and a random effect part β∈Rp2\mathbf{\beta}\in \mathbb{R}^{p_2}β∈Rp2​ the distribution of which is N(0,σβ2Ip2)\mathcal{N}(\mathbf{0}, \sigma^2_\mathbf{\beta} \mathbf{I}_{p_2})N(0,σβ2​Ip2​​), i.e., γT=[ωT,βT]\mathbf{\gamma}^T = [\mathbf{\omega}^T, \mathbf{\beta}^T]γT=[ωT,βT]. Then, we have
y=X1ω+X2β+e\begin{equation} \mathbf{y} = \mathbf{X}_1\mathbf{\omega} + \mathbf{X}_2\mathbf{\beta} + \mathbf{e} \end{equation}y=X1​ω+X2​β+e​​
Thus, we obtain the LMM (Linear Mixed-Effect Model). In this case, the number of parameters to estimate is p1+1+1p_1 + 1 + 1p1​+1+1 (including σβ2\sigma_\beta^2σβ2​ and σe2\sigma_e^2σe2​), which is much smaller than p+1p+1p+1 in the OLM.

Model Description

LMM is a statistical model that accounts for both fixed effects and random effects in a linear regression model. It is used for modeling data where observations are not independent or identically distributed.

Consider a dataset {y,X,Z}\{\mathbf{y}, \mathbf{X},\mathbf{Z}\}{y,X,Z} with nnn samples, where y∈Rn\mathbf{y} \in \mathbb{R}^ny∈Rn is the vector of response variable, X∈Rn×p\mathbf{X} \in \mathbb{R}^{n \times p}X∈Rn×p is the matrix of ppp independent variables, and Z∈Rn×c\mathbf{Z} \in \mathbb{R}^{n \times c}Z∈Rn×c is another matrix of ccc variables. The linear mixed model builds upon a linear relationship from y\mathbf{y}y to X\mathbf{X}X and Z\mathbf{Z}Z by
y=Zω⏟fixed+Xβ⏟random+e⏟error,\begin{equation} \mathbf{y} = \underbrace{\mathbf{Z}\mathbf{\omega}}_{\text {fixed}} + \underbrace{\mathbf{X}\mathbf{\beta}}_{\text {random}} + \underbrace{\mathbf{e}}_{\text {error}}, \end{equation}y=fixedZω​​+randomXβ​​+errore​​,​​
where ω∈Rc\mathbf{\omega} \in \mathbb{R}^cω∈Rc is the vector of fixed effects, β∈Rp\mathbf{\beta} \in \mathbb{R}^pβ∈Rp is the vector of random effects with β∼N(0,σβ2Ip)\mathbf{\beta} \sim \mathcal{N}(\mathbf{0}, \sigma^2_\mathbf{\beta} \mathbf{I}_p)β∼N(0,σβ2​Ip​), and e∼N(0,σe2In)\mathbf{e} \sim \mathcal{N}(\mathbf{0}, \sigma^2_e \mathbf{I}_n)e∼N(0,σe2​In​) is the independent noise term.

The LMM can be solved by various methods, such as the restricted maximum likelihood (REML) and the maximum likelihood (ML). The REML is a method of estimation that does not base estimates on a maximum likelihood fit of all the information, but instead uses a likelihood function derived from a transformed set of data, so that nuisance parameters have no effect. The ML is a method of estimating the parameters of a statistical model given observations, by finding the parameter values that maximize the likelihood of making the observations given the parameters. The ML is a special case of the maximum a posteriori estimation (MAP) that assumes that the prior over the parameters is uniform or non-informative.

Formal Description of Bayesian Inference and Variational Inference

The following contents are basically from Wikipedia-BI and Wikipedia-VBM. The purpose of this section is to provide a formal description of Bayesian inference and variational inference, which will be used in the following sections.

Definitions

  • x\mathbf{x}x: a data point in general.
  • X\mathbf{X}X: sample, a set of nnn observed data points, i.e., X={x1,x2,⋯ ,xn}\mathbf{X} = \{\mathbf{x}_1, \mathbf{x}_2, \cdots, \mathbf{x}_n\}X={x1​,x2​,⋯,xn​}.
  • x~\tilde{\mathbf{x}}x~: a new data point to be predicted.
  • θ\mathbf{\theta}θ: the parameters of the data point's distribution, i.e., x∼p(x∣θ)\mathbf{x} \sim p(\mathbf{x}|\mathbf{\theta})x∼p(x∣θ).
  • α\mathbf{\alpha}α: the hyperparameters of the distribution of θ\mathbf{\theta}θ, i.e., θ∼p(θ∣α)\mathbf{\theta} \sim p(\mathbf{\theta}|\mathbf{\alpha})θ∼p(θ∣α).
  • Z\mathbf{Z}Z: the latent variables of the data point's distribution, i.e., x∼p(x∣Z)\mathbf{x} \sim p(\mathbf{x}|\mathbf{Z})x∼p(x∣Z).
  • q(Z)q(\mathbf{Z})q(Z): variational distribution, a distribution over the latent variables Z\mathbf{Z}Z, i.e., P(Z∣X)≈q(Z)P(\mathbf{Z}|\mathbf{X}) \approx q(\mathbf{Z})P(Z∣X)≈q(Z).

Bayesian Inference

  • Prior: p(θ∣α)p(\mathbf{\theta}|\mathbf{\alpha})p(θ∣α), the distribution of the parameters θ\mathbf{\theta}θ before observing the data X\mathbf{X}X.
  • Sampling Distribution and Likelihood: p(X∣θ)=L(θ∣X)p(\mathbf{X}|\mathbf{\theta})=L(\mathbf{\theta}|\mathbf{X})p(X∣θ)=L(θ∣X), the distribution of the data X\mathbf{X}X given the parameters θ\mathbf{\theta}θ, or the likelihood of the parameters θ\mathbf{\theta}θ given the data X\mathbf{X}X.
  • Marginal Likelihood (Evidence): p(X∣α)p(\mathbf{X}|\mathbf{\alpha})p(X∣α), the distribution of the observed data X\mathbf{X}X marginalized over the prior distribution of the parameters θ\mathbf{\theta}θ, i.e., p(X∣α)=∫p(X∣θ)p(θ∣α)dθp(\mathbf{X}|\mathbf{\alpha}) = \int p(\mathbf{X}|\mathbf{\theta})p(\mathbf{\theta}|\mathbf{\alpha})d\mathbf{\theta}p(X∣α)=∫p(X∣θ)p(θ∣α)dθ.
  • Posterior: p(θ∣X,α)p(\mathbf{\theta}|\mathbf{X}, \mathbf{\alpha})p(θ∣X,α), the distribution of the parameters θ\mathbf{\theta}θ after observing the data X\mathbf{X}X. It is proportional to the product of the likelihood and the prior, i.e., p(θ∣X,α)∝p(X∣θ)p(θ∣α)p(\mathbf{\theta}|\mathbf{X}, \mathbf{\alpha}) \propto p(\mathbf{X}|\mathbf{\theta})p(\mathbf{\theta}|\mathbf{\alpha})p(θ∣X,α)∝p(X∣θ)p(θ∣α).
  • Bayes' Rule: p(θ∣X,α)=p(X∣θ)p(θ∣α)p(X∣α)∝p(X∣θ)p(θ∣α)p(\mathbf{\theta}|\mathbf{X}, \mathbf{\alpha}) = \frac{p(\mathbf{X}|\mathbf{\theta})p(\mathbf{\theta}|\mathbf{\alpha})}{p(\mathbf{X}|\mathbf{\alpha})}\propto p(\mathbf{X}|\mathbf{\theta})p(\mathbf{\theta}|\mathbf{\alpha})p(θ∣X,α)=p(X∣α)p(X∣θ)p(θ∣α)​∝p(X∣θ)p(θ∣α).
