In the previous section, we have applied the EM algorithm to the mixed effect model. In this section, we will use the MFVI algorithm to estimate the distributions of latent variables instead of using the MLE in E-step. And then, we will use the EM algorithm to estimate the parameters in M-step.

Variational Inference

Assume Θ={ω,σβ2,σe2}\Theta=\{\mathbf{\omega}, \sigma_\beta^2, \sigma_e^2\}Θ={ω,σβ2​,σe2​} has been estimated in the M-step. In order to use Mean-Field Variational Inference (MFVI) to find a q(β)q(\mathbf{\beta})q(β) which approximate the true posterior p(β∣y)p(\mathbf{\beta}|\mathbf{y})p(β∣y). We need to find the optimal q(β)q(\mathbf{\beta})q(β) that minimizes the KL divergence between q(β)q(\mathbf{\beta})q(β) and p(β∣y)p(\mathbf{\beta}|\mathbf{y})p(β∣y), which is equivalent to maximizing the ELBO (Evidence Lower BOund) function:
ELBO(q)=:Eq(β)[log⁡p(y,q(β))q(β)]=Eq(β)[log⁡p(y)]−KL(q(β)∣∣p(β))KL(q(β)∣∣p(β))=−Eq(β)[log⁡p(β∣y)q(β)]=−Eq(β)[log⁡p(β∣y)]+Eq(β)[log⁡q(β)]\begin{equation}\begin{split} \text{ELBO}(\mathbf{q}) &=: \mathbb{E}_{q(\mathbf{\beta})}[\log \frac{p(\mathbf{y}, q(\mathbf{\beta}))}{q(\mathbf{\beta})}] \\ &=\mathbb{E}_{q(\mathbf{\beta})}[\log p(\mathbf{y})] - \text{KL}(q(\mathbf{\beta})||p(\mathbf{\beta}))\\ \text{KL}(q(\mathbf{\beta})||p(\mathbf{\beta})) &= -\mathbb{E}_{q(\mathbf{\beta})}[\log \frac{p(\mathbf{\beta}|\mathbf{y})}{q(\mathbf{\beta})}] \\ &= -\mathbb{E}_{q(\mathbf{\beta})}[\log p(\mathbf{\beta}|\mathbf{y})] + \mathbb{E}_{q(\mathbf{\beta})}[\log q(\mathbf{\beta})] \end{split}\end{equation}ELBO(q)KL(q(β)∣∣p(β))​=:Eq(β)​[logq(β)p(y,q(β))​]=Eq(β)​[logp(y)]−KL(q(β)∣∣p(β))=−Eq(β)​[logq(β)p(β∣y)​]=−Eq(β)​[logp(β∣y)]+Eq(β)​[logq(β)]​​​

where KL(q(β)∣∣p(β))\text{KL}(q(\mathbf{\beta})||p(\mathbf{\beta}))KL(q(β)∣∣p(β)) is the KL divergence between q(β)q(\mathbf{\beta})q(β) and p(β∣y)p(\mathbf{\beta}|\mathbf{y})p(β∣y). And Eq(β)[log⁡p(y)]\mathbb{E}_{q(\mathbf{\beta})}[\log p(\mathbf{y})]Eq(β)​[logp(y)] is a constant with respect to q(β)q(\mathbf{\beta})q(β).

EM Algorithm with MFVI

Using the following algorithm to estimate E(β)\mathbb{E}(\mathbf{\beta})E(β) in the

  1. Initialize Θ(0)\Theta^{(0)}Θ(0) and μ(0)\mathbf{\mu}^{(0)}μ(0) randomly.
  2. For t=0,1,…t = 0, 1, \dotst=0,1,…, MAX_ITERATION:
    1. E-step: Estimate β^(t)\hat{\mathbf{\beta}}^{(t)}β^​(t).
    2. Compute ELBO\text{ELBO}ELBO
    3. M-step: Estimate Θ^(t+1)={ω^(t+1),σ^β2(t+1),σ^e2(t+1)}\hat{\Theta}^{(t+1)}=\{\hat{\mathbf{\omega}}^{(t+1)}, \hat{\sigma}_\beta^{2(t+1)}, \hat{\sigma}_e^{2(t+1)}\}Θ^(t+1)={ω^(t+1),σ^β2(t+1)​,σ^e2(t+1)​}.
    4. Compute Δℓc\Delta \ell_cΔℓc​. Check ∣Δℓc∣|\Delta \ell_c|∣Δℓc​∣ for convergence. If converged, stop. Otherwise, continue.
  3. Return results.
