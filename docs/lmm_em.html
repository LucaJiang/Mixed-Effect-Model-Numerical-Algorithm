Model Description

The linear mixed-effect model (LMM) is a statistical model that accounts for both fixed effects and random effects in a linear regression model. It is used for modeling data where observations are correlated and the correlations are not explained by the covariates.

Consider a dataset {y,X,Z}\{\mathbf{y}, \mathbf{X},\mathbf{Z}\}{y,X,Z} with nnn samples, where y∈Rn\mathbf{y} \in \mathbb{R}^ny∈Rn is the vector of response variable, X∈Rn×p\mathbf{X} \in \mathbb{R}^{n \times p}X∈Rn×p is the matrix of ppp independent variables, and Z∈Rn×c\mathbf{Z} \in \mathbb{R}^{n \times c}Z∈Rn×c is another matrix of ccc variables. The linear mixed model builds upon a linear relationship from y\mathbf{y}y to X\mathbf{X}X and Z\mathbf{Z}Z by
y=Zω+Xβ+e,\begin{equation} \mathbf{y} = \mathbf{Z}\mathbf{\omega} + \mathbf{X}\mathbf{\beta} + \mathbf{e}, \end{equation}y=Zω+Xβ+e,​​
where ω∈Rc\mathbf{\omega} \in \mathbb{R}^cω∈Rc is the vector of fixed effects, β∈Rp\mathbf{\beta} \in \mathbb{R}^pβ∈Rp is the vector of random effects with β∼N(0,σβ2Ip)\mathbf{\beta} \sim \mathcal{N}(0, \sigma^2_\mathbf{\beta} \mathbf{I}_p)β∼N(0,σβ2​Ip​), and e∼N(0,σe2In)\mathbf{e} \sim \mathcal{N}(0, \sigma^2_e \mathbf{I}_n)e∼N(0,σe2​In​) is the independent noise term. Let Θ\ThetaΘ denote the set of unknown parameters Θ={ω,σβ2,σe2}\Theta = \{\mathbf{\omega}, \sigma^2_\mathbf{\beta}, \sigma^2_e\}Θ={ω,σβ2​,σe2​}. We can treat β\mathbf{\beta}β as a latent variable because is it unobserved.

Complete Marginal Data Log-Likelihood

Variance of β\mathbf{\beta}β is known

Let Σ=σβ2XXT+σe2In\Sigma = \sigma_\mathbf{\beta}^2 \mathbf{X}\mathbf{X}^T + \sigma_e^2 \mathbf{I}_nΣ=σβ2​XXT+σe2​In​ be the covariance matrix of y\mathbf{y}y, then the conditional distribution of y\mathbf{y}y is
y∣Θ∼N(Zω,Σ)\mathbf{y}|\Theta \sim \mathcal{N}(\mathbf{Z}\mathbf{\omega}, \Sigma)y∣Θ∼N(Zω,Σ)

The complete data log-likelihood is given by
log⁡p(y∣Θ,β)=−12log⁡∣Σ∣−12(y−Zω)TΣ−1(y−Zω)+const\begin{equation} \begin{split} \log p(\mathbf{y}| \Theta, \mathbf{\beta}) &= -\frac{1}{2} \log |\Sigma| - \frac{1}{2} (\mathbf{y} - \mathbf{Z}\mathbf{\omega})^T \Sigma^{-1} (\mathbf{y} - \mathbf{Z}\mathbf{\omega}) + \text{const} \end{split}\end{equation}logp(y∣Θ,β)​=−21​log∣Σ∣−21​(y−Zω)TΣ−1(y−Zω)+const​​​

Variance of β\mathbf{\beta}β is unknown

Then the conditional distribution of y\mathbf{y}y become
y∣Θ,β∼N(Zω+Xβ,σe2In)\mathbf{y}|\Theta, \mathbf{\beta} \sim \mathcal{N}(\mathbf{Z}\mathbf{\omega}+ \mathbf{X}\mathbf{\beta}, \sigma_e^2\mathbf{I}_n)y∣Θ,β∼N(Zω+Xβ,σe2​In​)

The conditional log-likelihood is given by
log⁡p(y∣Θ,β)=−12log⁡∣σe2In∣−12(y−Zω−Xβ)T(σe2In)−1(y−Zω−Xβ)+const\begin{equation} \begin{split} \log p(\mathbf{y}| \Theta, \mathbf{\beta}) &= -\frac{1}{2} \log |\sigma_e^2 \mathbf{I}_n| - \frac{1}{2} (\mathbf{y} - \mathbf{Z}\mathbf{\omega} - \mathbf{X}\mathbf{\beta})^T (\sigma_e^2 \mathbf{I}_n)^{-1} (\mathbf{y} - \mathbf{Z}\mathbf{\omega} - \mathbf{X}\mathbf{\beta}) + \text{const} \end{split}\end{equation}logp(y∣Θ,β)​=−21​log∣σe2​In​∣−21​(y−Zω−Xβ)T(σe2​In​)−1(y−Zω−Xβ)+const​​​

E-Step

The E-step is to compute the expectation of the complete data log-likelihood with respect to the conditional distribution of β\mathbf{\beta}β given y\mathbf{y}y and Θ\ThetaΘ:
Q(Θ∣Θ(t))=Eβ∣y,Θ(t)[log⁡p(y,β∣Θ)]=Eβ∣y,Θ(t)[log⁡p(y∣β,Θ)+log⁡p(β∣Θ)]=Eβ∣y,Θ(t)[−12log⁡∣Σ∣−12(y−Zω)TΣ−1(y−Zω)+const]+Eβ∣y,Θ(t)[−12log⁡∣σβ2Ip∣−12βT(σβ2Ip)−1β+const]∝−Eβ∣y,Θ(t)[βT(σβ2Ip)−1β]\begin{equation} \begin{split} Q(\Theta|\Theta^{(t)}) &= \mathbb{E}_{\mathbf{\beta}|\mathbf{y}, \Theta^{(t)}} \left[ \log p(\mathbf{y}, \mathbf{\beta}| \Theta) \right] \\ &= \mathbb{E}_{\mathbf{\beta}|\mathbf{y}, \Theta^{(t)}} \left[ \log p(\mathbf{y}| \mathbf{\beta}, \Theta) + \log p(\mathbf{\beta}| \Theta) \right] \\ &= \mathbb{E}_{\mathbf{\beta}|\mathbf{y}, \Theta^{(t)}} \left[ -\frac{1}{2} \log |\Sigma| - \frac{1}{2} (\mathbf{y} - \mathbf{Z}\mathbf{\omega})^T \Sigma^{-1} (\mathbf{y} - \mathbf{Z}\mathbf{\omega}) + \text{const} \right] \\ &\quad + \mathbb{E}_{\mathbf{\beta}|\mathbf{y}, \Theta^{(t)}} \left[ -\frac{1}{2} \log |\sigma_\mathbf{\beta}^2 \mathbf{I}_p| - \frac{1}{2} \mathbf{\beta}^T (\sigma_\mathbf{\beta}^2 \mathbf{I}_p)^{-1} \mathbf{\beta} + \text{const} \right] \\ &\propto - \mathbb{E}_{\mathbf{\beta}|\mathbf{y}, \Theta^{(t)}} \left[ \mathbf{\beta}^T (\sigma_\mathbf{\beta}^2 \mathbf{I}_p)^{-1} \mathbf{\beta} \right] \end{split}\end{equation}Q(Θ∣Θ(t))​=Eβ∣y,Θ(t)​[logp(y,β∣Θ)]=Eβ∣y,Θ(t)​[logp(y∣β,Θ)+logp(β∣Θ)]=Eβ∣y,Θ(t)​[−21​log∣Σ∣−21​(y−Zω)TΣ−1(y−Zω)+const]+Eβ∣y,Θ(t)​[−21​log∣σβ2​Ip​∣−21​βT(σβ2​Ip​)−1β+const]∝−Eβ∣y,Θ(t)​[βT(σβ2​Ip​)−1β]​​​

Thus, the E-step is to compute the following expectations:

β^=(XTX+σe2σβ2Ip)−1XT(y−Zω)\begin{equation} \hat{\beta}= \left(\mathbf{X}^T \mathbf{X} + \frac{\sigma_e^2}{\sigma_\beta^2} \mathbf{I}_p \right)^{-1} \mathbf{X}^T (\mathbf{y} - \mathbf{Z}\mathbf{\omega}) %! \hat{\beta}= \left(\mathbf{X}^T \mathbf{X} + \frac{\sigma_e^2}{\sigma_\beta^2} \mathbf{I}_p \right)^{-1} \mathbf{X}^T \Sigma^{-1}(\mathbf{y} - \mathbf{Z}\mathbf{\omega}) \end{equation}β^​=(XTX+σβ2​σe2​​Ip​)−1XT(y−Zω)​​

And e^=y−Zω−Xβ^\begin{equation} \hat{e}= \mathbf{y} - \mathbf{Z}\mathbf{\omega} - \mathbf{X}\hat{\beta} \end{equation}e^=y−Zω−Xβ^​​​

M-Step

The M-step is to maximize the expectation of the complete data log-likelihood with respect to Θ\ThetaΘ:
Θ(t+1)=arg⁡max⁡ΘQ(Θ∣Θ(t))\begin{equation} \Theta^{(t+1)} = \arg\max_\Theta Q(\Theta|\Theta^{(t)}) \end{equation}Θ(t+1)=argΘmax​Q(Θ∣Θ(t))​​
which is equivalent to maximizing (2):
∂ℓ∂ω=ZTΣ−1(y−Zω)=:0⇒ω^=(ZTΣ−1Z)−1ZTΣ−1y\begin{equation} \begin{split} &\frac{\partial \ell}{\partial \mathbf{\omega}} = \mathbf{Z}^T \Sigma^{-1} (\mathbf{y} - \mathbf{Z}\mathbf{\omega})=:0\\ \Rightarrow \hat{\mathbf{\omega}} &= (\mathbf{Z}^T \Sigma^{-1} \mathbf{Z})^{-1} \mathbf{Z}^T \Sigma^{-1} \mathbf{y} \end{split}\end{equation}⇒ω^​∂ω∂ℓ​=ZTΣ−1(y−Zω)=:0=(ZTΣ−1Z)−1ZTΣ−1y​​​

Then, use Method of Moments(MoM) to estimate σβ2\sigma_\beta^2σβ2​ and σe2\sigma_e^2σe2​:
σ^β2=1p(E[βTβ]−E[β]2)σ^e2=1n(E[eTe]−E[e]2)\begin{equation} \begin{split} \hat{\sigma}_\beta^2 &= \frac{1}{p} \left(\mathbb{E}[\mathbf{\beta}^T\mathbf{\beta}] - \mathbb{E}[\mathbf{\beta}]^2 \right) \\ \hat{\sigma}_e^2 &= \frac{1}{n} \left(\mathbb{E}[\mathbf{e}^T\mathbf{e}] - \mathbb{E}[\mathbf{e}]^2 \right) \end{split}\end{equation}σ^β2​σ^e2​​=p1​(E[βTβ]−E[β]2)=n1​(E[eTe]−E[e]2)​​​

EM Algorithm

The EM algorithm is an iterative algorithm that alternates between the E-step and the M-step until convergence. The algorithm is summarized as follows:

  1. Initialize Θ(0)\Theta^{(0)}Θ(0).
  2. For t=0,1,…t = 0, 1, \dotst=0,1,…:
    1. E-step: Estimate β^(t)\hat{\mathbf{\beta}}^{(t)}β^​(t) and e^(t)\hat{\mathbf{e}}^{(t)}e^(t).
    2. M-step: Estimate Θ^(t+1)={ω^(t+1),σ^β2(t+1),σ^e2(t+1)}\hat{\Theta}^{(t+1)}=\{\hat{\mathbf{\omega}}^{(t+1)}, \hat{\sigma}_\beta^{2(t+1)}, \hat{\sigma}_e^{2(t+1)}\}Θ^(t+1)={ω^(t+1),σ^β2(t+1)​,σ^e2(t+1)​}.
    3. Check for convergence. If converged, stop. Otherwise, go to step 2.
  3. Return result.
